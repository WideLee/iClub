iClub
=====

----------------------------
*  一个社团学生互动平台    *
----------------------------

目前来说， 很多社团活动的宣传手段单一，难以起到理想的宣传效果，同时由于海报的设计不理想，很多学生在看海报时是一扫而过，很多关键信息被遗失，归根结底是因为没有一个统一的社团活动发布平台，不能让各个社团的最新活动消息集中有效的展示在同学面前，同时纸质版宣传不进有一定成本，而且盲目，没有针对性，例如以发传单的方式进行宣传时，不感兴趣的同学往往随手一丢，又造成了环境的破坏。因此以往的宣传在诸多方面都存在着很多问题，iClub 应运而生。
